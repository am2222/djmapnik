djmapnik
--------

Easy creation of Mapnik objects within GeoDjango.


Requires
--------

 * Mapnik (>= 0.7.0)
 * GeoDjango using PostGIS backend
 * Mapnik (>= 2.1.x) for MemoryDatasource support


Usage
-----

See the examples/geoadmin/README.txt for some usage.
